# Estructuras de Datos en Python

Implementacion de estructuras de datos fundamentales en Python, desarrollada originalmente en Google Colab como practica de algoritmos y estructuras.

## Que incluye

- Queue (cola FIFO) y Stack (pila LIFO)
- Lista enlazada simple
- Arbol binario
- Tabla hash

## Como usarlo

```bash
git clone https://github.com/epinki07/estructuras-datos-python.git
cd estructuras-datos-python

# En Jupyter
jupyter notebook notebooks/01_queue.ipynb

# Tests
python -m pytest tests/
```

## Tech Stack

Python 3.9+. El material original fue trabajado en Google Colab y Jupyter Notebook.
